A simple To-Do List application built with **React** and **TypeScript**.

How to run the project:
  1.	Clone the repository
    git clone https://github.com/YOUR_USERNAME/todo-list-app.git
  2.	Install Dependencies
    npm install
  3.	Start the development server
    npm run dev
  4.	Open your browser and go to http://localhost:3000/

Features:
  - Add, delete, and toggle tasks between completed and pending.
  - Tasks are saved in local storage, so they persist even after page refresh.
  - Filter tasks by All, Completed, or Pending.
  - Loading animation on page load.
  - Shows a message when there are no tasks.
  - Prevents adding empty tasks with a popup notification.

Technologies:
  - React
  - TypeScript
  - Tailwind CSS
